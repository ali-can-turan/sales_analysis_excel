🚛 Sales Operations Analysis: Strategic BI Dashboard

From Raw Data to Executive Decision Support
 
🎯 Project Overview

This project uses a raw data set that have been received in an education session, specifically, in the course "Excel for Business Skills: Advanced" by "Macquarie University" on "www.coursera.org". The raw data set have been used in an exam in the course, in which data analysis has NOT been expected by the exam taker. 
This project converts a raw "Sales Resources" dataset into an interactive Business Intelligence (BI) tool designed for operational efficiency and workforce planning.

The raw data basically includes sales amount of a store in October 2017 per salesperson and per product category: "books, clothing, games and music".
The Project "October 2017 Sales Breakdown" aims to answer two questions:

•	What are general insights about October 2017 sales amount per salesperson, per product category, per date, per transaction amount. (See "Summary - Analysis" section below.)

•	What could be done to increase efficiency? (See "Insights" section below.)

🛠 Technical Workflow & Data Integrity

A rigorous data cleaning process was implemented to ensure 100% accuracy before analysis:

•	Cleaning: Applied CLEAN, TRIM, and SUBSTITUTE functions to standardize text data.

•	Validation: Managed empty rows, removed duplicates, and enforced Data Validation for consistent entry.

•	Processing: Utilized MATCH / INDEX for date enrichment and logical mapping of sales representatives to respective categories.

•	UI/UX: Engineered an "application-like" experience using Scroll Bars and Data Validation lists instead of standard slicers for more granular control.

📊 Key Business Insight

The analysis revealed critical operational paradoxes and opportunities for optimization:

•	Volume vs. Value Paradox: While the Music category drives the highest transaction volume (frequency), it contributes the least to the total amount. Conversely, Books represent high-value transactions despite lower frequency. Nevertheless, Games category emerges with the highest value despite having minimum transaction volume.

💡 Strategic BI Options (Efficiency Planning)

Leveraging my background in Workforce Planning and Logistics Operations, I proposed two distinct organizational models based on the data findings:

•	Option A: Expertise-Based Planning

o	Focus: Assigning supervisors to specific product categories (e.g., B for Books, M for Music).

o	Pros: Deep domain expertise and faster processing times.

o	Cons: Risk of rigidity during personnel turnover.

•	Option B: Country/Account-Based Planning

o	Focus: Cross-functional supervisors managing two categories (one non-music + one music).

o	Pros: Medium domain expertise with medium operational flexibility.

o	Cons: Requires more intensive cross-training (Multi-Skilling).
